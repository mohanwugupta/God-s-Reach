TASK: Quick verification of extracted parameters (Batch).

INPUTS
- STUDY META: Study Type: ${study_type}; Experiments: ${num_experiments}
- EXTRACTED PARAMETERS (to check): ${parameter_list}
- CONTEXT (paper/methods/code): ${context}

INSTRUCTIONS
For each parameter name/value pair:
1) Scan CONTEXT. If the value matches, set "verified": true and include a very short supporting quote (≤150 chars).
2) If the value is WRONG, set "verified": false and provide:
   - "value": <correct value>,
   - "evidence": short quote (≤200 chars),
   - "reasoning": ≤1 short sentence.
3) If the parameter is NOT present in context, set "abstained": true and value: null.
4) Be concise. No explanations unless correcting an error.
5) JSON ONLY.

OUTPUT (strict JSON):
